# Connect-4
connect 4 online multi-player game

# Server

1. NodeJS
2. Socket.io
3. Express

## Install Dependency

```bash
cd connect-4-mo-server
npm install
```

## Run Server:

```bash
cd connect-4-mo-server
npm run start
```

# Client

1. React
2. Socket.io

## Install Dependency

```bash
cd connect-4-mo-client
npm install
```

## Run Dev Server:

```bash
cd connect-4-mo-client
npm run start
```

## Build For Production

```bash
cd connect-4-mo-client
npm run build
```
